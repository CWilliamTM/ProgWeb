A Linguagem HTML

HTML (abreviação para expressão inglesa HyperText Markup Language, que significa Linguagem de marcação de Hipertexto), é uma linguagem de marcação utilizada na construção de páginas na Web. Ducomentos HTML podem ser interpretados por navegadores.

Até este momento, vimos os seguintes recursos da linguagem:

Recurso Tags associadas Descrição
Links <a> Criar links entre as páginas
Listas <ul> e <li> Criar listas de itens ordenados
<ol> e <li> Criar listas de itens não ordenados
Imagens <img>
 Inserir imagens
